Python与C/C++相互调用

1.Python调用C、c++动态链接库

1.1 Python调用C库比较简单，不经过任何封装打包成so（linux） dll(windows)，再使用python的ctypes调用即可。

windows: 1）弄清楚编译的时候函数的调用约定采用的cdecl还是stdcall，因为根据DLL中函数调用约定方式，Python将使用相应的函数加载DLL。 2）如果采用C++的工程，那么导出的接口需要extern “C”，这样python中才能识别导出的函数。 3）dll（64位还是32位）要与python版本一致 vs2017 dll测试代码

#include <stdio.h>  
#include <stdlib.h>  

__declspec(dllexport) int foo(int a, int b);
__declspec(dllexport) int foo(int a, int b)
{
    printf("you input %d and %d\n", a, b);
    return a + b;
}

python 调用代码

import ctypes   
lib = ctypes.cdll.LoadLibrary("D:\\Algorithm\\python_other\\tipsTest\\Project5.dll")    
lib.foo(1, 3)  
print('***finish***')

Linux

#include <stdio.h>  
#include <stdlib.h>  
int foo(int a, int b)  
{  
  printf("you input %d and %d\n", a, b);  
  return a+b;  
}   

import ctypes  
ll = ctypes.cdll.LoadLibrary   
lib = ll("libpycall.so")    
lib.foo(1, 3)  
print('***finish***')

注：c依赖库生成命令： gcc -o libpycall.so -shared -fPIC pycall.c c++ 依赖库生成命令： g++ -o libpycall.so -shared -fPIC pycall.c
